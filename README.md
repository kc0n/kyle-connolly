# Kyle Connolly — Portfolio (GitHub Pages)

This folder is a complete, static portfolio site (HTML/CSS/JS). It is designed to be deployed on GitHub Pages.

## 1) Replace images (required)
Put your image files in: `assets/img/` and use these filenames:

- `profilepic.jpg`  (hero portrait)
- `fullstack.jpg`   (what I do section)
- `toblerone.jpg`
- `trustford.jpg`
- `ernestjones.jpg`
- `hsamuel.jpg`
- `seedsman.jpg`

If your files are PNG/WebP, either:
- rename them to `.jpg`, or
- update the file extension in `index.html`.

## 2) Add your CV (required)
Place your PDF in: `assets/cv/` and name it:

- `Kyle-Connolly-CV.pdf`

## 3) Publish via GitHub Pages
1. Create a new GitHub repository (Public), e.g. `kyle-portfolio`.
2. Upload **all** files and folders from this ZIP into the repository root.
3. In GitHub: **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** / Folder: **/(root)**
6. Save and wait ~1 minute.

Your site will be live at:
`https://YOUR-USERNAME.github.io/REPO-NAME/`
